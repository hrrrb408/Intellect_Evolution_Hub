# IEH Soul

## For future Claude
This file defines the long-term purpose and operating character of this vault.

## Mission

IEH exists to become an intellectual operating system: a place where source
material becomes durable understanding, durable understanding becomes research
direction, and research direction becomes action.

The vault should not merely store files. It should help the user think better.

## Core beliefs

- A source is not fully ingested until it changes the processed layer.
- A summary is not enough; good notes should teach.
- Query pages are first-class knowledge objects.
- AI tools are interchangeable clients. The vault protocol is the center.
- Raw sources must remain traceable so future reasoning can be audited.
- Automation is useful only when it is conservative, observable, and reversible.

## What IEH should become

IEH should support:

- paper reading and synthesis;
- technical learning;
- project memory;
- personal strategy and decisions;
- reusable concepts and comparisons;
- long-horizon learning maps;
- cross-tool continuity between Codex, Claude, Hermes, and future agents.

## What IEH should avoid

- Becoming a file dump.
- Treating generated reports as final knowledge.
- Letting command prompts diverge across runtimes.
- Allowing automation to silently overwrite human judgment.
- Over-claiming from small source batches.

## North star

Every important addition should make IEH more useful for one of three things:

1. answering a question;
2. learning a field;
3. deciding what to do next.
